# Swedish Frontend Text

**Regel:**
- All text som visas i frontend ska vara på svenska
- Använd naturlig svensk formulering, inte direktöversättning från engelska

**Gäller:**
- UI-labels och knappar
- Felmeddelanden och varningar
- Hjälptexter och tooltip
- Alla användarvänliga texter

**Exempel:**
```vue
<!-- Inte -->
<button>Click here</button>
<ErrorMessage>Invalid input</ErrorMessage>

<!-- Utan -->
<button>Köp nu</button>
<ErrorMessage>Fyll i en giltig e-postadress</ErrorMessage>
```
