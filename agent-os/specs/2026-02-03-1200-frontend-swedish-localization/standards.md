# Standards: Frontend Swedish Localization

## Applicabla Standards

### accessibility.md
- Översättningar ska inte påverka semantisk HTML
- placeholder-texter ska översättas
- ARIA-labels ska uppdateras vid behov

### components.md
- Översättningar ska vara konsekventa inom samma komponent
- Om samma text används på flera ställen, använd samma översättning

## Externa Standarder

### Svenska språkkonventioner
- Använd svenska datumformat (YYYY-MM-DD eller sv-SE locale)
- Valuta: "kr" eller "SEK" (kr används i befintlig kod)
- "öre" som enhet för ören
